1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, where each node corresponds to part of the document, such as elements, attributes, and text.

2. How do you select an element from the DOM?

You can select elements from the DOM using several methods in JavaScript. Here are some of the most common techniques: 1. getElementById 2. getElementsByClassName 3. getElementsByTagName 4. querySelector 5. querySelectorAll

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element instead of individual listeners to each child element. This allows the parent to handle events that bubble up from its child elements.

4. How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using various properties and methods in the DOM.
